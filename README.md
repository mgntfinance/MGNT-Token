> ## 🏗️ Protocol Overview
>
> ### Core Mechanics
>
> Magnet Finance operates on a self-reinforcing flywheel:
>
> ```
> Swap Volume → Protocol Fees → Buybacks & Burns → Scarcity
>      ↑                                                 ↓
> Capital Attraction ←←←← Value Accrual ←←←← Reduced Supply
> ```
>
> Every mechanic within the protocol is designed to compound. Swap volume drives burns. Staking drives scarcity. Scarcity drives value. Value attracts capital. Capital deepens liquidity.
